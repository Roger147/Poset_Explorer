## Concepts Clarified

- Matching size counts chain links, not chains.
- If a chain cover has `n` elements and `m` matched links, it has `n - m`
  chain components.
- Width can be found through the minimum chain-cover size.
- A chain can contain at most one element from any antichain, which gives the
  intuitive lower bound connecting antichains and chain covers.
- Transitive comparability does not need to be explicitly stored in the input
  relations, but algorithms such as width must account for it internally.
- Family factories generate examples; analyzers measure posets.
